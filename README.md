# Coastal Blue Digital

Official website and source code for Coastal Blue Digital.

---

## Overview

Coastal Blue Digital is a professional website project built to provide affordable web design services for local service businesses.

The goal of this project is to create a modern, responsive website while documenting the development process using Git and GitHub.
